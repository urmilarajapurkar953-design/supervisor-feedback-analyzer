# Supervisor Feedback Analyzer — Trinethra Module

A full-stack AI-assisted dashboard engineered for psychology interns to process supervisor text transcripts smoothly using local LLMs. It breaks down walls of raw conversation into structured, editable rubric metrics while actively fighting automation bias.

---

## 🚀 Setup & Installation Instructions

### 1. Prerequisites
Ensure you have **Node.js** (v18+) and **Ollama** installed on your system.

### 2. Set Up the Local LLM Engine
Open a brand-new terminal window and pull the target lightweight model:
```bash
ollama pull llama3.2
